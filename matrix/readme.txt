Установка либы:
pip install numpy
можно в корневой интерпретатор, она полезная.
Скрипты зациклены, после получения результата можно вводить новые матрицы.
Матрицы вводятся построчно, числа в строке разделяются пробелами, после каждой строки жмем Enter.
Когда закончили вводить матрицу, пишем end и жмем Enter.

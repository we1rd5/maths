Установка либы:
pip install numpy
можно в корневой интерпретатор, она полезная.
Скрипты зациклены, после получения результата можно вводить новые матрицы.
Матрицы вводятся построчно, числа в строке разделяются пробелами, после каждой строки жмем Enter.
Когда закончили вводить матрицу, пишем end и жмем Enter.
Что делает конкретный скрипт можно понять из названия или в комментах в самом коде.
Все вычисления производит numpy, поэтому багов в бэке быть не должно.
UI и ввод матрицы писал я, никаких try-except естественно нет, говнокод присутствует, баги должны быть.
Новые функции пишу параллельно с прохождением курса, поэтому можно следить за апдейтами.
